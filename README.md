This is a [Next.js](https://nextjs.org/) playground project for Next.js 12 and React 18

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
